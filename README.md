# FX Multi Pair Dashboard

為替レートAPIを利用し、複数通貨ペアを自動監視して
エントリーシグナル（BUY / SELL / 待機）を判定するダッシュボードアプリです。

自動売買ではなく **監視＆手動エントリー運用** のため安全に使用できます。


## ⚙ 機能一覧

| 機能 | 状態 |
|------|------|
| 無料の為替レートAPIを利用 | ✔ |
| チャート生成（15分足相当） | ✔ |
| BUY / SELL / 待機の自動判定 | ✔ |
| 強さ評価 ★1〜★5 | ✔ |
| 通貨一覧を強さ順ソート表示 | ✔ |
| クリックでチャート展開 | ✔ |
| 通貨別パラメータ編集 & 保存 | ✔ |
| ブラウザUI（Streamlit） | ✔ |

---

## 🚀 必要なもの

- Python 3.9+
- インターネット接続（為替レート取得のため）
